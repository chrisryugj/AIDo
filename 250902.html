📋시작하기 전에 (필독)

⏰ 소요 시간: 총 20~30분 (각 실습 5~10분)

💻 준비물: PC 또는 모바일 브라우저, 메모앱

🔒 보안수칙: 실제 개인정보나 기밀정보 입력 금지, 가명·예시 데이터만 사용

1핵심 용어 이해하기

🎯 기본 개념

  • 생성형 AI: 대량의 데이터를 학습하여 새로운 텍스트나 이미지를 생성하는 고급 자동완성 시스템
  • LLM(대규모 언어모델): 방대한 텍스트를 학습한 언어 모델 (예: ChatGPT, Claude)
  • 프롬프트: AI에게 주는 명령어. 역할, 맥락, 과업, 형식, 제약, 검증 요소를 포함하면 정확도 향상

🔬 고급 개념

  • 역할 지시(Role): "당신은 공공문서 편집자입니다"처럼 AI의 관점을 설정
  • 샷 기법: 예시를 통해 원하는 스타일 학습 (Zero-shot: 예시 없음 / One-shot: 예시 1개 / Few-shot: 예시 여러 개)
  • 토큰: AI가 처리하는 최소 단위. 너무 길면 중요 내용이 누락될 수 있음
  • 환각(Hallucination): AI가 사실과 다른 정보를 그럴듯하게 생성하는 오류
  • 컨텍스트 창: AI가 한 번에 참고할 수 있는 정보량 한계

2AI 작동 원리 (3분 이해)

입력(프롬프트) → AI 모델(패턴 분석) → 출력(응답) → 검증·수정 → 재입력

핵심 원칙:

  • 좋은 입력이 좋은 출력을 만든다
  • 검증과 수정 과정을 거치면 품질이 크게 향상된다

3프롬프트 작성법 (R-C-T-F-C-V 공식)

📝 6단계 구성 요소

  • R(Role): 전문가 역할 지정
  • C(Context): 상황, 대상, 목표 설명
  • T(Task): 구체적인 작업 내용
  • F(Format): 출력 형식 (표, 불릿, 문단, 글자수 등)
  • C(Constraints): 제약사항, 톤, 분량 등
  • V(Verification): "불확실하면 질문하고, 출처와 한계를 명시하세요"

아이디어 기대효과 필요자원

올바른 분리배출 퀴즈 재미와 학습을 결합한 참여 유도 온라인 설문폼, 소형 경품
